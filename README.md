# :floppy_disk: Installation

1. Clone the repo:
   ```sh
   git clone https://github.com/yuryiva/todo-list-simple.git
   ```
2. Install NPM packages:
   ```sh
   npm install
   ```
3. Run the app in the development mode with `npm start`
